### 登录

* url: `login`  
* method: `POST`
* request:

```json
{
    "username": "admin",
    "password": "admin"
}
```

* response:

```json
{
    "token": "Token 22591812540bd094f279992db3320fafb017faa3",
    "groups": []
}
```
